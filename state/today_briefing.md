# Daily AI Recruitment Briefing — 3 July 2026

## Claude & Anthropic in TA

- **Anthropic shipped Claude Sonnet 5**, an agent-focused model priced at an introductory $2/$10 per million input/output tokens (through 31 August, then $3/$15) — the drop makes autonomous screening, sourcing, and outreach agents that were cost-prohibitive at flagship rates routine for high-volume TA teams. Note the new tokenizer can map the same input to ~1.0–1.35× more tokens, so re-baseline any per-candidate cost models. ([Anthropic](https://www.anthropic.com/news/claude-sonnet-5)) (30 June)
- **Anthropic released Claude Tag in beta for Enterprise and Team users in Slack**, giving everyone in a company a single shared Claude "identity" so employees can hand off half-finished tasks — a pattern worth piloting for collaborative req intake and candidate hand-offs across a hiring team. ([Fortune](https://fortune.com/2026/06/23/anthropic-claude-tag-virtual-employee-tool-slack/)) (23 June)

## AI in Recruitment — What's New

- **LinkedIn's Hiring Assistant is now live in French and German**, its first expansion beyond English; pilot users reportedly saved 4+ hours per role and reviewed 62% fewer profiles. If you run multilingual req pipelines in EMEA, this is the moment to test whether the agent's sourcing quality holds up outside English. ([StartupHub.ai](https://www.startuphub.ai/ai-news/tech/2026/linkedin-hiring-assistant-goes-global)) (23 June)

**Bottom line:** With frontier agent models now cheap enough to run at applicant-pool scale and recruiter-facing AI agents maturing across languages, TA leaders should stand up governed pilots of agentic sourcing/screening now — and re-check vendor cost assumptions against the new, cheaper token economics.
