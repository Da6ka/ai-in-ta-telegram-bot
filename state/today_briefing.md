# Daily AI Recruitment Briefing — 4 July 2026

## Claude & Anthropic in TA
- **Anthropic launched Claude Tag, an always-on Claude that lives inside Slack** as a shared team agent that breaks down tasks, uses connected tools, and builds context over time — in beta for Team and Enterprise plans. The catch for TA teams already using the old Claude-in-Slack app: migration is mandatory and that app retires 3 August, so admins must opt in and scope channel access, tool connections, and memory before the deadline. ([Fortune](https://fortune.com/2026/06/23/anthropic-claude-tag-virtual-employee-tool-slack/)) (23 June)
- **Anthropic is reportedly preparing a Claude agent for Microsoft Teams**, extending the same workplace-agent push behind Claude Tag into Microsoft's collaboration suite. If it ships, TA orgs standardized on Teams get a native path to Claude-based workflows — worth watching before committing to a Slack-only agent rollout. ([WinBuzzer](https://winbuzzer.com/2026/07/01/anthropic-eyes-claude-agent-for-microsoft-teams-xcxwbn/)) (1 July)

## AI in Recruitment — What's New
- **Mega HR launched Megan, billed as the first fully functional AI recruiter running directly inside Greenhouse**, handling screening, outreach, scheduling, interview documentation, and candidate communication — added in minutes with no implementation fees, and with continuous bias/risk auditing on its evaluations. For Greenhouse shops, this is a low-friction way to test an agentic recruiter against your live pipeline; pressure-test its audit trail before letting it touch candidate-facing decisions. ([PR Newswire](https://www.prnewswire.com/news-releases/greenhouse-gets-its-first-ai-recruiter-meet-megan-your-always-on-hiring-partner-302585222.html)) (21 June)

**Bottom line:** The agentic layer is landing where recruiters already work — inside Slack, Teams, and the ATS itself — so the near-term TA priority is governance (access scope, migration deadlines, bias auditing) rather than tool selection.
